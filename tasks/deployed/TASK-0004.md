## Problem

Several pages, sections, and references need to be removed or updated:

1. **Impressum page** (`/impressum`): Should be removed entirely from the site.
2. **"Co říkají lékaři" (testimonials) section**: Should be hidden for now; will be re-enabled after more doctor feedback is collected.
3. **`anote.cz` links and mentions**: All references to `anote.cz` must be removed from the site. The custom domain is not yet live, so linking to it is incorrect.
4. **Hero/page headline**: The phrase `Lékařské zprávy z hlasu` should be replaced with `Lékařské zprávy automaticky`.

## Desired Outcome

- The Impressum page is removed. Any navigation links, footer links, or sitemap entries pointing to `/impressum` are removed.
- The "Co říkají lékaři" section is not rendered (can be commented out or conditionally hidden). The section should be easy to re-enable later.
- No occurrences of `anote.cz` remain in the rendered output, links, metadata, or configuration files (except in internal documentation `.md` files in the ANOTE-web repo — those are out of scope).
- Any configuration or metadata field that currently uses `anote.cz` as a domain (canonical URLs, Open Graph `og:url`, hreflang `href`, Plausible Analytics domain) is updated to use the current temporary Azure hostname: `https://yellow-forest-086a45303.7.azurestaticapps.net`. **This is a temporary value** — it must be replaced with the final production domain when it is ready. Add a `// TODO: replace with production domain` comment wherever this placeholder is used in code.
- The headline `Lékařské zprávy z hlasu` is replaced with `Lékařské zprávy automaticky` everywhere it appears.

## Constraints

- The Impressum page removal must also be reflected in the sitemap (`src/app/sitemap.ts`) and any navigation/footer components.
- The `anote.cz` removal must include checking `staticwebapp.config.json`, `next.config.ts`, SEO metadata, Open Graph tags, hreflang tags, and dictionary files — not just visible text.
- The "Co říkají lékaři" section must be hidden in a way that makes it easy to restore (prefer a `hidden` flag or conditional render over deleting the component).
- The headline replacement must be applied in both Czech and English dictionaries.
